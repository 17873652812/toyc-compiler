## 六、ToyC 语言定义

### 6.1 概述

- C 语言的简化子集
- **省去**：数组、指针、I/O、多文件编译
- ToyC 源文件可用 C 编译器直接编译，运行结果与等同 C 代码一致

### 6.2 数据类型

- `int` — 32 位有符号整数
- `void` — 函数返回类型

### 6.3 关键字

`int` `void` `const` `if` `else` `while` `break` `continue` `return`

## 七、ToyC 文法（BNF 产生式）

**开始符号**：`CompUnit`

```
编译单元    CompUnit    → (Decl | FuncDef)⁺

声明        Decl        → ConstDecl | VarDecl

常量声明    ConstDecl   → "const" "int" ID "=" Expr ";"

变量声明    VarDecl     → "int" ID "=" Expr ";"

语句        Stmt        → Block
                        | ";"
                        | Expr ";"
                        | ID "=" Expr ";"
                        | Decl
                        | "if" "(" Expr ")" Stmt ("else" Stmt)?
                        | "while" "(" Expr ")" Stmt
                        | "break" ";"
                        | "continue" ";"
                        | "return" Expr? ";"

语句块      Block       → "{" Stmt* "}"

函数定义    FuncDef     → ("int" | "void") ID "(" (Param ("," Param)*)? ")" Block

形参        Param       → "int" ID

表达式      Expr        → LOrExpr

逻辑或      LOrExpr     → LAndExpr | LOrExpr "||" LAndExpr

逻辑与      LAndExpr    → RelExpr | LAndExpr "&&" RelExpr

关系        RelExpr     → AddExpr
                        | RelExpr ("<" | ">" | "<=" | ">=" | "==" | "!=") AddExpr

加减        AddExpr     → MulExpr | AddExpr ("+" | "-") MulExpr

乘除模      MulExpr     → UnaryExpr | MulExpr ("*" | "/" | "%") UnaryExpr

一元        UnaryExpr   → PrimaryExpr | ("+" | "-" | "!") UnaryExpr

基本        PrimaryExpr → ID | NUMBER | "(" Expr ")"
                        | ID "(" (Expr ("," Expr)*)? ")"
```

## 八、ToyC 终结符特征

### 8.1 标识符 (ID)

- 符合 C 规范的标识符
- 正则：`[_A-Za-z][_A-Za-z0-9]*`

### 8.2 整数 (NUMBER)

- 十进制整数常量
- 正则：`0|[1-9][0-9]*`（NUMBER 不含负号，负数由 UnaryExpr 的 `-` 统一处理）

### 8.3 空白字符和注释

- **空白字符**：空格、制表符、换行符等 — 忽略
- **单行注释**：`//` 开始，到最近的换行符前结束
- **多行注释**：`/*` 开始，到最近的 `*/` 结束

## 九、ToyC 语义约束

### 9.1 程序结构

- 可包含全局变量声明、全局常量声明、函数定义
- **必须包含** `int main()` 函数作为入口点

### 9.2 函数

- 参数：若干 `int` 类型参数
- 返回值：`int` 或 `void`
- 只能声明在**全局作用域**，名称唯一
- 不能作为值来储存、传递或运算
- 函数调用必须写在被调函数**声明之后**（支持递归）
- **int 函数**：每条执行路径必须 return 一个 int 值
- **void 函数**：可无 return；有则不能带返回值

### 9.3 语句

- 语法语义与 C 语言一致
- 无返回值的函数调用不能作为 if/while 条件或赋值右值
- break / continue 只能出现在循环中

### 9.4 变量与常量声明

- 支持 32 位有符号整数 (int) 类型的局部/全局变量、局部/全局常量 (const)
- **初始化要求**：所有声明必须有初始化表达式
- **常量约束**：
  - const 标识符为常量，值在编译期确定
  - 初始化表达式只能包含数字字面量、已声明的常量及其算术/逻辑运算
  - 声明后不可修改（不能作为赋值左值）
- **作用域**：
  - 全局变量/常量：整个程序运行期间有效
  - 局部变量/常量：声明点到作用域结束
  - 形参：函数体内可见
  - Block 创建嵌套作用域，内层屏蔽外层同名符号
  - 变量/常量使用必须在声明之后

### 9.5 表达式

- 支持逻辑运算、关系运算、算术运算、括号、变量引用、字面值、函数调用
- 运算符优先级、结合性、计算规则与 C 一致
- `&&` 和 `||` 遵循短路计算
- 非零为真、零为假
- 除数不能为零
